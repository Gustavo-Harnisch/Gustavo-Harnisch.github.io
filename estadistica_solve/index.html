EJERCICIOS RESUELTOS (Del Listado 04)
Ejercicio 1:
En una población, P(T) = 0,70 (tuberculino-positivo) y P(R) = 0,60 (anticuerpos contra rubéola). Si ambos sucesos son independientes, calcule P(T o R).
Solución:
Fórmula: P(T ∪ R) = P(T) + P(R) - P(T ∩ R)
Como son independientes: P(T ∩ R) = P(T) · P(R) = 0,70 × 0,60 = 0,42
P(T ∪ R) = 0,70 + 0,60 - 0,42 = 0,88
Respuesta: P(T o R) = 0,88 o 88%
Ejercicio 2:
De 50 vendedores que recibieron bono, 20 asistieron a capacitación. Empresa con 200 empleados. Sea B = bono, S = capacitación. Hallar P(B), P(S|B) y P(B ∩ S).
Solución:
P(B) = 50/200 = 0,25
P(S|B) = 20/50 = 0,40 (20 de 50 bonificados asistieron)
P(B ∩ S) = P(B) · P(S|B) = 0,25 × 0,40 = 0,10
Respuesta: P(B) = 0,25 ; P(S|B) = 0,40 ; P(B ∩ S) = 0,10
Ejercicio 3:
P(diagnóstico correcto) = 0,7. P(demanda | diagnóstico incorrecto) = 0,9. ¿Cuál es P(diagnóstico incorrecto Y demanda)?
Solución:
P(incorrecto) = 1 - 0,7 = 0,3
Fórmula: P(incorrecto ∩ demanda) = P(incorrecto) · P(demanda|incorrecto)
P(incorrecto ∩ demanda) = 0,3 × 0,9 = 0,27
Respuesta: 0,27 o 27%
Ejercicio 4 (Talca):
P(helado) = 60%, P(torta) = 36%, P(torta|helado) = 40%. Calcular P(helado|torta).
Solución (Bayes):
P(torta ∩ helado) = P(helado) · P(torta|helado) = 0,60 × 0,40 = 0,24
P(helado|torta) = P(torta ∩ helado) / P(torta) = 0,24 / 0,36 = 2/3 ≈ 0,667
Respuesta: P(helado|torta) ≈ 0,667 o 66,7%
Ejercicio 5:
Máquina A: 3000 envases, 2% defectuosos. Máquina B: 2000 envases, 4% defectuosos. a) P(defectuoso) b) P(de A | defectuoso)? P(de B | defectuoso)?
Solución (Ley Prob. Total y Bayes):

Parte a)

P(A) = 3000/5000 = 0,6 ; P(B) = 2000/5000 = 0,4
P(def|A) = 0,02 ; P(def|B) = 0,04
P(def) = P(A)·P(def|A) + P(B)·P(def|B) = 0,6×0,02 + 0,4×0,04 = 0,012 + 0,016 = 0,028

Parte b)

P(A|def) = [P(A)·P(def|A)] / P(def) = (0,6×0,02) / 0,028 = 0,012/0,028 ≈ 0,429
P(B|def) = [P(B)·P(def|B)] / P(def) = (0,4×0,04) / 0,028 = 0,016/0,028 ≈ 0,571
Respuesta: a) P(def) = 0,028 ; b) P(A|def) ≈ 0,429 ; P(B|def) ≈ 0,571
Ejercicio 6:
P(A) = 0,5, P(B) = 0,3, P(C) = 0,1. A y B independientes; B y C mutuamente excluyentes. Calcular: a) B y C ambos b) Solo uno de A ó B c) A dado que B
Solución:

Parte a) B ∩ C

Como B y C son mutuamente excluyentes: P(B ∩ C) = 0

Parte b) Solo uno de A ó B = (A ∩ Bᶜ) ∪ (Aᶜ ∩ B)

P(A ∩ Bᶜ) = P(A)·(1-P(B)) = 0,5 × 0,7 = 0,35 (independientes)
P(Aᶜ ∩ B) = (1-P(A))·P(B) = 0,5 × 0,3 = 0,15
P(solo uno) = 0,35 + 0,15 = 0,50

Parte c) P(A|B)

Como A y B son independientes: P(A|B) = P(A) = 0,5
Respuesta: a) 0 ; b) 0,50 ; c) 0,5
Ejercicio 7:
P(Talca) = 0,7, P(Curicó) = 0,4, P(Talca ∪ Curicó) = 0,8. ¿P(ambas ciudades)? ¿P(ninguna)?
Solución:

Parte a) P(Talca ∩ Curicó)

Fórmula: P(T ∪ C) = P(T) + P(C) - P(T ∩ C)
0,8 = 0,7 + 0,4 - P(T ∩ C)
P(T ∩ C) = 1,1 - 0,8 = 0,3

Parte b) P(ninguna) = P((T ∪ C)ᶜ)

P((T ∪ C)ᶜ) = 1 - P(T ∪ C) = 1 - 0,8 = 0,2
Respuesta: a) 0,3 ; b) 0,2
Ejercicio 8:
Distribución de clientes en fondos: 20% dinero, 15% corto, 10% intermedio, 5% largo, 18% alto riesgo, 25% moderado, 7% balanceado. a) P(balanceado) b) P(bonos) c) P(no acciones)
Solución:
a) P(balanceado) = 0,07 = 7%
b) P(bonos) = 0,15 + 0,10 + 0,05 = 0,30 = 30%
c) P(acciones) = 0,18 + 0,25 = 0,43 ; P(no acciones) = 1 - 0,43 = 0,57 = 57%
Respuesta: a) 7% ; b) 30% ; c) 57%
Ejercicio 9:
Tres proyectos: P(A₁)=0,22, P(A₂)=0,25, P(A₃)=0,28, P(A₁∩A₂)=0,11, P(A₁∩A₃)=0,05, P(A₂∩A₃)=0,07, P(A₁∩A₂∩A₃)=0,01. Calcular: a) P(A₁∪A₂) b) P(A₁ᶜ∩A₂ᶜ) c) P(A₁∪A₂∪A₃)
Solución:

Parte a) P(A₁ ∪ A₂)

P(A₁ ∪ A₂) = P(A₁) + P(A₂) - P(A₁ ∩ A₂) = 0,22 + 0,25 - 0,11 = 0,36

Parte b) P(A₁ᶜ ∩ A₂ᶜ) = P((A₁ ∪ A₂)ᶜ)

P((A₁ ∪ A₂)ᶜ) = 1 - P(A₁ ∪ A₂) = 1 - 0,36 = 0,64

Parte c) P(A₁ ∪ A₂ ∪ A₃)

= 0,22 + 0,25 + 0,28 - 0,11 - 0,05 - 0,07 + 0,01 = 0,53
Respuesta: a) 0,36 ; b) 0,64 ; c) 0,53
Ejercicio 10:
A = no excede mínimo en enero, B = no excede en julio. P(A)=0,8, P(B)=0,7, P(A∪B)=0,9. a) P(A∩B) b) P(exceso exactamente uno de dos meses)
Solución:

Parte a) P(A ∩ B)

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
0,9 = 0,8 + 0,7 - P(A ∩ B)
P(A ∩ B) = 1,5 - 0,9 = 0,6

Parte b) Exceso exactamente uno = (Aᶜ ∩ B) ∪ (A ∩ Bᶜ)

P(Aᶜ ∩ B) = P(B) - P(A ∩ B) = 0,7 - 0,6 = 0,1
P(A ∩ Bᶜ) = P(A) - P(A ∩ B) = 0,8 - 0,6 = 0,2
P(exactamente uno) = 0,1 + 0,2 = 0,3
Respuesta: a) 0,6 ; b) 0,3
Ejercicio 11:
A = solicitud sobre SPSS, B = solicitud sobre SAS. P(A)=0,30, P(B)=0,50. a) ¿Por qué P(A)+P(B)≠1? b) P(Aᶜ) c) P(A∪B) d) P(Aᶜ∩Bᶜ)
Solución:
a) Porque existen otras solicitudes posibles (no son complementarios ni exhaustivos)
b) P(Aᶜ) = 1 - 0,30 = 0,70
c) Suponiendo A y B mutuamente excluyentes: P(A ∪ B) = 0,30 + 0,50 = 0,80
d) P(Aᶜ ∩ Bᶜ) = 1 - P(A ∪ B) = 1 - 0,80 = 0,20 (otras solicitudes)
Respuesta: a) Otras solicitudes posibles ; b) 0,70 ; c) 0,80 ; d) 0,20
Ejercicio 12 (Conteo):
5 profesores (Anderson, Box, Cox, Cramer, Fisher) seleccionar 2 para comité. a) P(Anderson Y Box) b) P(al menos un miembro con C) c) P(≥15 años experiencia)
Solución:
Total de pares: C(5,2) = 10

Parte a) Anderson Y Box

Solo 1 forma de elegir este par
P(Anderson y Box) = 1/10 = 0,1

Parte b) Al menos un C (Cox o Cramer)

P(ningún C) = C(3,2)/C(5,2) = 3/10
P(al menos un C) = 1 - 3/10 = 7/10 = 0,7

Parte c) ≥15 años (3,6,7,10,14 años). Pares con ≥15:

(7,10): 17✓, (7,14): 21✓, (10,14): 24✓, (6,10): 16✓, (6,14): 20✓
Pares válidos: 5 ; P = 5/10 = 0,5
Respuesta: a) 0,1 ; b) 0,7 ; c) 0,5
Ejercicio 13 (Demostración):
Demostrar que si A ⊆ B, entonces P(A) ≤ P(B)
Demostración:
Si A ⊆ B, entonces B = A ∪ (B ∩ Aᶜ) (union disjunta)
P(B) = P(A) + P(B ∩ Aᶜ) (aditividad, eventos disjuntos)
Como P(B ∩ Aᶜ) ≥ 0 (axioma 2): P(B) ≥ P(A)
Por lo tanto: P(B - A) = P(B) - P(A)
Ejercicio 14:
P(verde)=0,09, P(blanco)=0,15, P(rojo)=0,21, P(azul)=0,23. ¿P(uno de esos colores)?

RESUMEN DE FÓRMULAS CLAVE

Probabilidad Básica

Complemento
P(Aᶜ) = 1 - P(A)
Unión
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
Condicional
P(A|B) = P(A ∩ B) / P(B) , P(B) > 0

Teorema de Bayes

Ley Probabilidad Total
P(E) = Σᵢ P(Aᵢ) · P(E|Aᵢ)
Bayes
P(Aᵢ|E) = P(Aᵢ) · P(E|Aᵢ) / Σⱼ P(Aⱼ) · P(E|Aⱼ)
